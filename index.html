Latar Belakang

Saya telah bekerja pada salah satu software house di Yogyakarta selama sekitar 1 tahun lebih 3 bulan dan startup di jakarta selama 7 bulan. Selama itu saya telah belajar tentang banyak tentang ilmu programming mulai dari analisa masalah, desain sistem, implementasi, testing hingga maintenance.

Berikut daftar bahasa pemrograman yang telah saya pelajari:

  1. HTML + CSS (Front-end)
  2. PHP (Engine)
  3. Laravel, CI, Yii (Framework)
  4. Javascript, jQuery (interface)
  5. MySQL (database)

Saya juga telah memahami beberapa istilah istilah ataupun penggunaan dalam pembuatan programing seperti MVC, REST API, Cron Job, AJAX, Responsive Web, GIT repository dan lainnya. Walau masih sedikit yang saya kuasai, namun saya selalu berminat untuk selalu belajar ilmu pemrograman.

Menurut saya semua bahasa pemrograman hampir sama, hanya berbeda dalam fungsi dan metode penggunaannya. Secara dasar yang diperlukan dalam menguasasi pemrograman adalah referensi dan logika.. Pengetahuan referensi yang baik akan memudahkan untuk menemukan logika yang tepat.

Dalam proses developing kita harus menguasai tiap bagian dasar yang menjadi pondasi daripada aplikasi tersebut. Untuk itu saya jauh lebih menyukai untuk mempelajari setiap komponen pemrograman dari yang paling awal atau dari pembangunnya yang juga biasa disebut scratch. Karena ini akan jauh lebih memudahkan untuk penyusunan dan maintenance ke depannya. Pembuatan konsep program sebenarnya jauh lebih penting daripada pembuatan kode program itu sendiri

Minat

Untuk saat ini saya tertarik untuk belajar dan mengembangkan bahasa pemrograman Javascript. Karena fungsionalitas dan kecepatannya yang sangat baik sekali dan bisa digunakan hampir di semua environment browser.

Hanya saja bahasa ini memerlukan penguasaan yang tidak mudah daripada bahasa pemrograman internet lainnya. Terutama karena struktur penulisan dan metode yang dipakai masih sangat menyulitkan para developer.

Namun berkat kerja para developer akhirnya terdapat suatu plugin yang memudahkan dalam menggunakan code javascript layaknya library seperti misalkan jQuery.

Pengembangan dari javascript sendiri selalu jauh lebih cepat ketimbang bahasa pemrograman lainnya. Baru baru ini muncul teknologi server side dari Javascript yang dibungkus dalam satu paket bernama nodeJS. Dan juga telah muncul beberapa framework yang makin memudahkan dalam pembuatan web berbasis script-server seperti ReactJS, Angular, dan kawan kawannya.

Visi dan misi

Visi dan misi dalam hidup saya cukup sederhana, hanya ingin memajukan dan mengembangkan teknologi informasi dan turut membantu memenuhi kebutuhan orang lain khususnya dalam bidang teknologi informasi berbasis internet.

Untuk dapat memenuhi visi dan misi saya, maka saya memerlukan suatu wadah yang mana dapat menampung dan mengembangkan karya saya. Bukan kantor dan tugas yang saya cari namun rekan kerja dan saling "sharing" yang saya perlukan.

Banyak orang mengira bahwa manusia itu bekerja untuk dirinya sendiri, atau bekerja karena membutuhkan materi yang akan digunakan nantinya. Padahal sebenarnya tidak, siklus alami yang terjadi adalah manusia membantu orang lain dalam memenuhi kebutuhannya. Karena sejak manusia dilahirkan pasti akan selalu ada kebutuhan yang harus dipenuhi. Itulah sebabnya mengapa saya tidak terlalu mementingkan masalah gaji/upah, karena setiap apa yang saya terima telah sesuai dengan kemampuan saya dalam memenuhi kebutuhan orang lain. Jadi, para pembaca sekalian, mari kita bekerja sama dan saling membantu :D.
