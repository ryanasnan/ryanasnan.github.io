JQuery Selector¶

Jquery telah menyediakan cara untuk memilih suatu element DOM berdasarkan dari selector-nya.

Jika dikelompokkan maka akan terdapat 4 tipe selector yang digunakan yaitu common selector, select by position of element, select by element attribute, input selector.

Tabel dari masing masing selector dan reference nya.

Common Selector

name_selector selector example explain
All Element
$(“*”) Menyeleksi semua element pada html
Select by Identifier #id $(“#identifier”) Menyeleksi element yang memiliki id=”identifier”
Select by Class .class $(“.class”) Menyeleksi element yang memiliki class=”class”
Select by Element name-of-element $(“p”) Menyeleksi element <p>
Multi Element Selection selector1, selector2, … , selectorn $(“h1,div,p”) Menyeleksi element <h1>, <div> dan <p>
Direct Element Selection selector-parent > selector-child $(“div > p”) Menyeleksi element <p> yang mana memiliki parent dalam satu level berupa element <div>
Parent Descendant selector1 selector 2 $(“div p”) Menyeleksi element <p> yang mana memiliki parent dalam beberapa level berupa element <div>
Next Element Selection selector1 + selector-next $(“div + p”) Menyeleksi element <p> yang berada setelah elament <div>
Element Sibling selector1 ~ selector-sibling $(“div ~ p”) Menyeleksi element <p> yang berada pada level yang sama dengan element <div> dan berada setelah element <div> tersebut
Multiple attribute selector in one Element selector1selector 2 $(“div.class”) Menyeleksi element <div> yang memiliki class=class

Select By Position of Element

name_selector selector example explain
Position Element :first $(“p:first”) Menyeleksi element <p> pertama (awal)
Position Element :last $(“p:last”) Menyeleksi element <p> terakhir
Position Element :even $(“tr:even”) Menyeleksi element <tr> yang berindex genap (0,2,4), namun jika dilihat seperti ganjil (dimulai dari element awal atau pertama) ini dikarenakan index element selalu dimulai dari 0
Position Element :odd $(“tr:odd”) Menyeleksi element <tr> yang berindex ganjil (1,3,5), namun jika dilihat seperti genap (dimulai dari element kedua) ini dikarenakan index element selalu dimulai dari index 0
Specific Position Element :first-child $(“p:first-child”) Menyeleksi element <p> yang berada di index awal (0) pada tiap terdapat adanya element parent, namun jika pada awal element child adalah selain <p>, maka ketika setelahnya terdapat element <p> tetap tidak akan terseleksi
Specific Position Element :first-of-type $(“p:first-of-type”) Menyeleksi element <p> yang berada di index awal (0) pada tiap terdapat adanya element parent, ini berlaku pada tiap child element yang memiliki element <p>
Specific Position Element :last-child $(“p:last-child”) Menyeleksi element <p> yang berada di index terakhir pada tiap terdapat adanya element parent, namun jika pada akhir element child adalah selain <p>, maka ketika sebelumnya terdapat element <p> tetap tidak akan terseleksi
Specific Position Element :last-of-type $(“p:last-of-type”) Menyeleksi element <p> yang berada di index terakhir pada tiap terdapat adanya element parent, ini berlaku pada tiap child element yang memiliki element <p>
Specific Position Element :nth-child(n) $(“p:nth-child(2)”) Menyeleksi element <p> yang berada di index 2 pada tiap terdapat adanya element parent, namun jika pada index 2 element child adalah selain <p>, maka ketika setelahnya terdapat element <p> tetap tidak akan terseleksi
Specific Position Element :nth-last-child(n) $(“p:nth-last-child(2)”) Menyeleksi element <p> yang berada di index 2 dihitung dari bawah pada tiap terdapat adanya element parent, namun jika pada index tersebut element child adalah selain <p>, maka ketika sebelumnya terdapat element <p> tetap tidak akan terseleksi
Specific Position Element :nth-of-type(n) $(“p:nth-of-type(2)”) Menyeleksi element <p> yang berada di index 2 pada tiap terdapat adanya element parent, ini berlaku pada tiap child element yang sama dan berada di index 2 yang memiliki element <p>
Specific Position Element :nth-last-of-type(n) $(“p:nth-last-of-type(2)”) Menyeleksi element <p> yang berada di index 2 dihitung dari bawah pada tiap terdapat adanya element parent, ini berlaku pada tiap child element yang sama dan berada di index 2 dihitung dari bawah yang memiliki element <p>
Specific Position Element :only-child $(“p:only-child”) Menyeleksi element <p> yang sendirian pada suatu parent. namun jika terdapat element lain sebelumnya maka tidak akan terseleksi
Specific Position Element :only-of-type $(“p:only-of-type”) Menyeleksi element <p> yang sendirian pada suatu parent element. Ini juga berlaku jika semisal terdapat element lain pada parent tersebut
Single Index Selection :eq(index) $(“p:eq(3)”) Menyeleksi element <p> pada index 3. Namun ini tidak berpengaruh ketika ada element <p> lain yang berada di suatu child dan berada pada index 3
Multi Index Selection :gt(no) $(“ul li:gt(3)”) Menyeleksi element <li> (pada parent ul) yang dimulai dari index 3 dan akan berlaku pada element selanjutnya yang sama (element <li>)
Multi Index Selection :lt(no) $(“ul li:lt(3)”) Menyeleksi element <li> (pada parent ul) yang dimulai dari index 3 dihitung dari bawah dan akan berlaku pada element diatasnya yang sama (element <li>)

Select by Element Attribute

name_selector selector example explain
Select by Attribute [attribute] $(“[href]”) Menyeleksi element yang memiliki attribute “href”. Ini juga bisa dikombinasikan dengan “attribute=value”
Has Selection :has(selector) $(“div:has(p)”) Menyeleksi element <div> yang memiliki element <p>
Contain Selection :contains(text) $(“p:contains(is)”) Menyeleksi element <p> yang memiliki text berupa “is”
Visibility Selection :empty $(“:empty”) Menyeleksi element yang kosong, misal adalah column table yang tidak ada isinya
Parent Selection :parent $(“td:parent”) Menyeleksi element <td> yang tidak kosong, ini merupakan kebalikan dari selector :empty
Visibility Selection :hidden $(“p:hidden”) Menyeleksi element <p> yang tersembunyi, atau yang memiliki atribute “style:display:none”
Visibility Selection :visible $(“p:visible”) Menyeleksi element <p> yang terlihat atau tidak tersembunyi
Not Selection :not(selector) $(“p:not(.intro)”) menyeleksi element <p> yang tidak memliki class “intro”

Input Selector

name_selector selector example explain
Input Type :input $(“:input”) Menyeleksi semua element input
Input Type :text $(“:text”) Menyeleksi element input dengan attribute type=”text”
Input Type :password $(“:password”) Menyeleksi element input dengan attribute type=”password”
Input Type :radio $(“:radio”) Menyeleksi element input dengan attribute type=”radio”
Input Type :checkbox $(“:checkbox”) Menyeleksi element input dengan attribute type=”checkbox”
Input Type :submit $(“:submit”) Menyeleksi element input dengan attribute type=”submit”
Input Type :reset $(“:reset”) Menyeleksi element input dengan attribute type=”reset”
Input Type :button $(“:button”) Menyeleksi element input dengan attribute type=”button”
Input Type :image $(“:image”) Menyeleksi element input dengan attribute type=”image”
Input Type :file $(“:file”) Menyeleksi element input dengan attribute type=”file”
Input variety Element :enabled $(“:enabled”) Menyeleksi element input yang enable (tidak sedang disable)
Input variety Element :disabled $(“:disabled”) Menyeleksi element input dyang disabled
Input variety Element :selected $(“:selected”) Menyeleksi element input yang dipilih
Input variety Element :checked $(“:checked”) Menyeleksi element input yang dicek
